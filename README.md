
# Kelvin Kariuki - Digital Portfolio



A modern, responsive portfolio website showcasing my skills, projects, and professional journey as a UI/UX Designer and Web Developer. Built with pure HTML, CSS, and JavaScript.

## 🔥 Features
- Vibrant colors and sleek animations
- **Responsive Design**: Works on mobile, tablet, and desktop
- **Interactive Elements**:
  - Project cards with hover effects
  - Star rating system
  - Smooth scrolling navigation
- **Dynamic Sections**:
  - About me with tabbed content
  - Project showcase grid
  - Contact form with validation

## 🛠️ Technologies Used
- **Frontend**: HTML5, CSS3, JavaScript
- **Styling**: CSS Variables (for theme colors), Flexbox/Grid
- **Icons**: Font Awesome 6
- **Hosting**: <Netlify.app

## 🚀 Installation
just open https://myportofolio-kelvin.netlify.app/ in your browser.
No installation required! Open  `index.html` directly in your browser.
